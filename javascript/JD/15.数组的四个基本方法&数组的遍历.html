数组的四个基本方法:
push() 向数组的最后面插入一个或多个元素，返回结果为该数组新的长度-会改变原数组、数组的新长度 = 数组.push(元素);
pop() 删除数组中的最后一个元素，返回结果为被删除的元素-会改变原数组、被删除的元素 = 数组.pop();
unshift() 在数组最前面插入一个或多个元素，返回结果为该数组新的长度-会改变原数组、数组的新长度 = 数组.unshift(元素);
shift() 删除数组中的第一个元素，返回结果为被删除的元素-会改变原数组、被删除的元素 = 数组.shift();

for循环
forEach() 和 for循环类似，但需要兼容IE8以上 forEach() 没有返回值。也就是说，它的返回值是 undefined
arr.forEach(function(item, index, obj){}
回调函数中传递三个参数：
  第一个参数，就是当前正在遍历的元素
  第二个参数，就是当前正在遍历的元素的索引
  第三个参数，就是正在遍历的数组
filter() 返回结果是true的项，将组成新的数组。可以起到过滤的作用 不会改变原数组
var arr1 = [1, 3, 6, 2, 5, 6];
var arr2 = arr1.filter(function (item, index) {
   return item > 4; //将arr1中大于4的元素返回
})
map() 对原数组中的每一项进行加工
var arr1 = [1, 3, 6, 2, 5, 6];
var arr2 = arr1.map(function (item, index) {
   return item + 10; //让arr1中的每个元素加10
})
every() 如果有一项返回false，则停止遍历 意思是，要求每一项都返回true，最终的结果才返回true
every()方法的返回值是boolean值，参数是回调函数。
some() 只要有一项返回true，则停止遍历
